Isolation Mechanisms & Speciation

Mekanisme isolasi reproduksi adalah penghalang yang mencegah aliran gen antar populasi dan menjadi dasar terbentuknya spesiasi.

Diagram Konsep:
Prezygotic → Habitat / Temporal / Behavioral / Mechanical / Gametic
Postzygotic → Inviability / Sterility / Hybrid Breakdown

1. Isolasi Pra-Zigot

  • Habitat isolation: Populasi menempati habitat berbeda sehingga tidak saling bertemu.
  • Temporal isolation: Perbedaan musim atau waktu reproduksi.
  • Behavioral isolation: Ritual kawin spesifik.
  • Mechanical isolation: Struktur genital tidak kompatibel.
  • Gametic isolation: Gamet tidak dapat fusi karena inkompatibilitas molekuler.

2. Isolasi Pasca-Zigot

  • Hybrid inviability: zigot/hibrid gagal berkembang.
  • Hybrid sterility: individu steril (cth: mule).
  • Hybrid breakdown: F1 bisa hidup, F2 rusak/lemah.

3. Mode Spesiasi

  • Allopatric: terpisah barrier geografis.
  • Peripatric: populasi kecil terisolasi di pinggiran.
  • Parapatric: hibrid zone tipis, seleksi berbeda.
  • Sympatric: tanpa isolasi geografis, biasanya karena poliploidi atau niche berbeda.
  • Quantum speciation: perubahan cepat karena reorganisasi kromosom besar.

Tutorial GeneStudio, BLAST & Identification Engine

Panduan analisis sekuens DNA mulai dari membaca kromatogram, cleaning, hingga identifikasi spesies.

Diagram Alur:
.ab1 → Editing → Consensus Sequence → (jika CDS: cek stop codon) → BLAST → Identifikasi BOLD

1. GeneStudio

  • Import file .ab1
  • Perbaiki peak noise, hapus ambiguitas (N)
  • Reverse complement bila diperlukan
  • Gabung forward–reverse menjadi contig

2. Pengecekan CDS

  • Translasi otomatis → pastikan tidak ada premature stop codon
  • Jika ada, ulangi editing hingga bersih

3. BLAST

  • Gunakan BLASTn khusus untuk DNA barcode
  • Perhatikan %identity, query cover, dan E-value

4. Identification Engine (BOLD)

  • Masukkan FASTA ke BOLD → Cocokkan species-level hits

Genetic Variation: Quantitative Variation

Variasi kuantitatif adalah sifat poligenik yang menghasilkan fenotipe kontinu.

Diagram Distribusi:
Bentuk kurva normal akibat kontribusi banyak gen kecil + lingkungan.
  • Dikendalikan banyak gen (polygenic)
  • Fenotipe kontinu: tinggi badan, biomassa, ukuran organ
  • Komponen variasi: Vp = Vg + Ve + Vge
  • Heritabilitas (h²) menentukan kemampuan sifat merespons seleksi

Genetic Variation: Molecular Variation

Mencakup variasi DNA pada level nukleotida.

Diagram: SNP → Polymorphic Sites → Haplotypes
  • SNP: variasi satu basa
  • Polymorphic sites: posisi genom dengan lebih dari satu alel
  • Haplotype: kombinasi karakter nukleotida unik per individu
  • Heterozygosity: peluang dua alel berbeda pada lokus tertentu

Genetic Variation: Chromosome Variation

Perubahan struktur atau jumlah kromosom.

Diagram Klasifikasi:
Aneuploidi | Poliploidi | Delesi | Duplikasi | Inversi | Translokasi
  • Aneuploidi: kehilangan/penambahan 1 kromosom (2n±1)
  • Poliploidi: 3n, 4n, 6n (umum di tanaman)
  • Inversi: segmen membalik
  • Delesi: segmen hilang
  • Duplikasi: segmen berulang
  • Translokasi: pertukaran antarkromosom
